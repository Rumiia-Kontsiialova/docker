Что такое Docker?

    b) Контейнерная платформа для упаковки, доставки и запуска приложений.

Соотнесите описание(поданы ниже) и понятие (Docker, Классическая виртуализация):

Каждая виртуальная машина имеет свою собственную операционную систему = Классическая визуализация 

Быстрый запуск и эффективное использование ресурсов = Docker

Все контейнеры на одном хосте используют общую операционную систему = Docker

Запускает полные копии операционной системы = Классическая виртуализация

Использует контейнеризацию для упаковки и запуски приложения = Docker

Использует гипервизор для виртуализации аппаратных ресурсов = Классическая виртуализация



Верно ли утверждение: Все контейнеры, запущенные на одной машине, делят ядро операционной системы, поэтому они стартуют немедленно и более эффективно.

    а) Да


Какой из компонентов Docker отвечает за выполнение и управление контейнерами?

    a) Docker Engine



К какому из нижеперечисленных компонентов Docker относится это описание: Изолированные среды, которые запускаются на основе образов и могут содержать всё необходимое для выполнения приложения.

    c) Docker Container


Что представляет собой Dockerfile?

    a) Текстовый файл, описывающий шаги по созданию Docker образа.



Что представляют собой Docker Images?

    b) Шаблоны для создания контейнеров, состоящие из слоев



Что такое Docker Hub и другие Docker реестры (registries)?

    b) Онлайн-платформа для обмена, хранения и управления Docker образами.
